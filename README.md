OVERVIEW - 

This project is a Streamlit-based web scraping application that extracts Pokémon statistics from PokemonDB and displays them in an interactive dashboard.


FEATURES-

Scrapes Pokémon data from PokemonDB
Displays Pokémon statistics in a table
Search and filter Pokémon by name
Interactive Streamlit interface


TECHNOLOGIES USEd -

Python
Streamlit
Requests
BeautifulSoup4
Pandas


DATA SOURCE-

https://pokemondb.net/pokedex/all
